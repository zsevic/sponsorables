# next-starter

> Minimal Next.js boilerplate

### Setup

```bash
git clone https://github.com/zsevic/next-starter
cd next-starter
cp .env.sample .env.local # change values after copying
npm i
npm run dev
```

### Build

```bash
npm run build
npm start
```

### Linting

```bash
npm run lint
npm run lint:fix
```

### Testing

```bash
npm test
```

### Technologies used

- React, Next.js
